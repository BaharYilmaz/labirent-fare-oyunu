# Labirent Fare Oyunu

## Amaç

Oyun, labirent içinde farenin peynire ulaşmasını amaçlar.

## Özelllikler

* Bilgisayar peyniri bulabilir.
* Oyuncu yön tuşlarıyla peynire ulaşabilir.
